StructDefinitionType extends DefinitionType

Represents a struct which contains a fixed set of defined properties
{
  "type": String,
  "parent": String,
  "base": Boolean,
  "properties": Map (PropertyType),
  "discriminator": String,
  "mapping": Map (String),
  "template": Map (String),
}
FieldDescription
typeString
parentString
The parent type of this struct. The struct inherits all properties from the parent type
baseBoolean
Indicates that this struct is a base type, this means it is an abstract type which is used by different types as parent
propertiesMap (PropertyType)
discriminatorString
In case this is a base type it is possible to specify a discriminator property
mappingMap (String)
In case a discriminator property was set it is possible to specify a mapping. The key is the type name and the value the concrete value which is mapped to the type
templateMap (String)
In case the parent type contains generics it is possible to set a concrete type for each generic type
